## Overview

The Sheet component provides a flexible slide-out panel that can appear from any side of the screen. It's built on Radix UI's Dialog primitive and is perfect for navigation menus, forms, settings panels, and additional content that doesn't require a full page.

## Features

- **Multiple Positions**: Slide from top, right, bottom, or left
- **Responsive Design**: Adapts to different screen sizes
- **Keyboard Navigation**: Full keyboard support with focus management
- **Backdrop Interaction**: Click outside to close (configurable)
- **Escape Key**: Close with Escape key
- **Focus Trapping**: Keeps focus within the sheet when open
- **Accessibility**: Complete ARIA support and screen reader compatibility
- **Customizable Content**: Flexible header, body, and footer sections

## API Reference

### Sheet

The root sheet component that manages the open/closed state.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `defaultOpen` | `boolean` | `false` | The initial open state |
| `open` | `boolean` | - | Controlled open state |
| `onOpenChange` | `(open: boolean) => void` | - | Callback when open state changes |

### SheetTrigger

Button that triggers the sheet to open.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `asChild` | `boolean` | `false` | Render as child element |

### SheetContent

The main content container of the sheet.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `side` | `"top" \| "right" \| "bottom" \| "left"` | `"right"` | Side of screen to slide from |
| `className` | `string` | - | Additional CSS classes |

### SheetHeader

Header section of the sheet, typically containing title and description.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |

### SheetTitle

Title element for the sheet header.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |

### SheetDescription

Description element for the sheet header.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |

### SheetFooter

Footer section of the sheet, typically containing action buttons.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |

### SheetClose

Button that closes the sheet when clicked.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `asChild` | `boolean` | `false` | Render as child element |

## Accessibility

The Sheet component includes comprehensive accessibility features:

- **Focus Management**: Automatically focuses the first focusable element when opened
- **Focus Trapping**: Keeps focus within the sheet while open
- **Keyboard Navigation**: Escape key closes the sheet, Tab cycles through focusable elements
- **ARIA Attributes**: Proper `role`, `aria-labelledby`, and `aria-describedby` attributes
- **Screen Reader Support**: Announces sheet opening/closing and content changes

## Usage Guidelines

### When to Use

- **Navigation Menus**: Mobile-friendly navigation panels
- **Forms**: Side panels for editing or creating content
- **Settings**: Configuration panels and preferences
- **Filters**: Advanced filtering options for data views
- **Details**: Additional information that doesn't require a full page

### When Not to Use

- **Critical Actions**: Use modals for important confirmations
- **Complex Workflows**: Multi-step processes might need full pages
- **Large Content**: Extensive content might be better on dedicated pages

## Examples

### Controlled Sheet

```tsx
import { Sheet, SheetContent, SheetTrigger } from '@/components/ui/sheet'
import { useState } from 'react'

function ControlledSheet() {
  const [open, setOpen] = useState(false)

  return (
    <Sheet open={open} onOpenChange={setOpen}>
      <SheetTrigger asChild>
        <Button>Open Sheet</Button>
      </SheetTrigger>
      <SheetContent>
        <p>This sheet is controlled by state</p>
        <Button onClick={() => setOpen(false)}>
          Close Programmatically
        </Button>
      </SheetContent>
    </Sheet>
  )
}
```
