## Overview

The Collapsible component provides a way to hide and show content with smooth animations. It's perfect for creating expandable sections, settings panels, and content that needs to be toggled on demand while maintaining a clean interface.

## Features

- **Smooth Animations**: Built-in expand and collapse animations with CSS transitions
- **Controlled/Uncontrolled**: Support for both controlled and uncontrolled usage patterns
- **Keyboard Navigation**: Full keyboard accessibility with Enter and Space key support
- **Flexible Trigger**: Any element can be used as a trigger with `asChild` prop
- **Accessibility**: ARIA attributes and screen reader support built-in
- **Customizable**: Easy to style and extend with additional functionality
- **Performance**: Optimized animations with minimal layout shifts

## Examples

### Basic Implementation

```tsx
import {
  Collapsible,
  CollapsibleContent,
  CollapsibleTrigger,
} from "@/components/ui/collapsible"
import { Button } from "@/components/ui/button"
import { ChevronDown, ChevronRight } from "lucide-react"
import { useState } from "react"

function BasicCollapsible() {
  const [isOpen, setIsOpen] = useState(false)

  return (
    <Collapsible open={isOpen} onOpenChange={setIsOpen}>
      <div className="flex items-center justify-between space-x-4">
        <h4 className="text-sm font-semibold">
          System Components
        </h4>
        <CollapsibleTrigger asChild>
          <Button variant="ghost" size="sm" className="w-9 p-0">
            {isOpen ? (
              <ChevronDown className="h-4 w-4" />
            ) : (
              <ChevronRight className="h-4 w-4" />
            )}
            <span className="sr-only">Toggle</span>
          </Button>
        </CollapsibleTrigger>
      </div>
      <div className="rounded-md border px-4 py-3 font-mono text-sm">
        @core/system
      </div>
      <CollapsibleContent className="space-y-2">
        <div className="rounded-md border px-4 py-3 font-mono text-sm">
          @core/components
        </div>
        <div className="rounded-md border px-4 py-3 font-mono text-sm">
          @core/utilities
        </div>
      </CollapsibleContent>
    </Collapsible>
  )
}
```

### With Custom Trigger

## Accessibility

- Full keyboard navigation with Enter and Space key support
- Proper ARIA attributes for screen readers (`aria-expanded`, `aria-controls`)
- Focus management maintains logical tab order
- Screen reader announcements for state changes

## Best Practices

1. **Clear Triggers**: Use descriptive text and visual indicators for triggers
2. **Smooth Animations**: Keep animations fast and smooth (avoid overly long transitions)
3. **Logical Grouping**: Group related content that benefits from being collapsible
4. **State Indicators**: Provide clear visual feedback for open/closed states
5. **Accessibility**: Always include screen reader text for icon-only triggers
